#### 이미지를 넣고 싶을 때
1. 프로젝트 내에 assets 폴더 만들고 이미지 파일을 거기 넣어둡니다
2. pubspec.yaml 파일에 assets 폴더 등록
3. main.dart에 이미지 넣기

```dart
MaterialApp(
  home: Image.asset('assets/dog.png')
)
```
